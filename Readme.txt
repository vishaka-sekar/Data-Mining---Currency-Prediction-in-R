Step 1:
Download and Install R Studio from https://www.rstudio.com/products/rstudio/download2/ by selecting the platform based on your machine configuration.
If the R-Studio is already installed kindly clear the global environment before starting the code.

Step 2:
Download the given project source code named "Team3P3.R" and Dataset named "USDINR.xlsx" .

Step 3:
Open Rstudio and open the given project source code.

Step 4: Build Instructions
Install the below given the library LINE BY LINE in the R-STUDIO CONSOLE
install.packages("EMD")
install.packages("readxl")
install.packages("rnn")
install.packages("forecast")
install.packages("gdata")
install.packages("tseries")
install.packages("XLConnect")
install.packages("xlsx")
install.packages("ModelMetrics")

Step 5:Verifying the library Installation
Make sure all the libraries are installed successfully without any error messages.

Step 6:
Select the source code from Line 13 which says "Start Running the code from below after successful installation of the libraries" 

Step 7:
Fix the Number of days based on the need which is given in line 25

Step 8:
Look for the Plotted Graphs on Lower Right pane to look for the graph prediction and console for the predicted value output with error prediction as well.
Also look into the video attached for the way to see multiple graphs.
